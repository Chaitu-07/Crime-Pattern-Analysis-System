# 📊 Crime Pattern Analysis System

A **Data Analytics Project** that analyzes crime datasets to identify trends, visualize crime patterns, and generate meaningful insights through interactive dashboards.

The project demonstrates the complete data analytics workflow—from data ingestion and cleaning to database storage, visualization, and geographic analysis.

---

# 📌 Project Overview

The **Crime Pattern Analysis System** is designed to help analyze crime data using modern data analytics techniques.

Crime datasets are uploaded as CSV files, cleaned and validated using **Python & Pandas**, stored in **PostgreSQL**, and analyzed through SQL queries. The results are displayed using interactive charts and maps, allowing users to explore crime trends and patterns efficiently.

This project focuses on practical data analysis rather than predictive machine learning, making it an ideal portfolio project for aspiring **Data Analysts**.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Crime-Pattern-Analysis-System.git
```

---

## Navigate to Project

```bash
cd Crime-Pattern-Analysis-System
```

---

## Create Virtual Environment

Windows

```bash
python -m venv venv
```

Activate

```bash
venv\Scripts\activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Configure PostgreSQL

Create a PostgreSQL database.

Update your database configuration inside:

```
config.py
```

---

## Run the Project

```bash
python app.py
```

---

## Open in Browser

```
http://127.0.0.1:5000
```
